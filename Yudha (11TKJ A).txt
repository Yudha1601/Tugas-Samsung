Soal :
1. Apa perbedaan file system Linux dan Windows?
2. Berikan gambar struktur file system Linux dan Windows?
3. Apa yang dimaksud direktori,folder,dan file?
4. Sebutkan 5 file extension dan jelaskan?
5. Sebutkan masing-masing 5 perintah mengakses file di Linux dan windows?
6. Apa perbedaan terminal dan command prompt?

Jawaban :
1. Pada Linux (Unix-like) semua file dan direktori “bercabang” mulai dari satu titik paling atas yang disebut “root” yaitu /, sedangkan di Windows struktur file system berbasis drive letter (contoh: C:\) dan punya folder sistem, folder pengguna, folder program, tiap drive bisa punya akar sendiri. 

2. Linux :
/
├─ bin/  (binaries)
├─ boot/
├─ dev/
├─ etc/  (config)
├─ home/ (user folders)
├─ lib/
├─ media/
├─ mnt/
├─ opt/
├─ proc/
├─ root/
├─ sbin/
├─ srv/
├─ tmp/
├─ usr/
└─ var/
Windows :
C:\
├─ Windows\
│  └─ System32\
├─ Program Files\
├─ Program Files (x86)\
├─ Users\
│  └─ <username>\
│     ├─ Documents\
│     ├─ Downloads\
│     └─ Desktop\
└─ Temp\

3. File = unit penyimpanan data; bisa teks, gambar, video, program, dll, kalo Folder (kadang kata lain “directory”) = tempat untuk mengelompokkan file atau folder lain; kayak map di dalam map, sedangkan Direktori = istilah yang sering dipakai di sistem Unix/Linux untuk “folder” (ya intinya sama: wadah yang punya entri file atau sub-direktori).

4. .txt - file teks biasa, cuma karakter tanpa format rumit.
    .exe - file executable di Windows, artinya program bisa dijalankan. 
    .docx - dokumen Microsoft Word (format dokumen). 
    .mp4 - file video, format multimedia. 
    .png - file gambar raster dengan kualitas bagus, biasanya mendukung transparansi

5. Linux (terminal)-- perintah akses/manipulasi file/folder:
ls - listing isi direktori. 
pwd - menunjukkan direktori kerja sekarang. 
cd - pindah direktori. 
cp - copy file atau folder. 
rm - hapus file atau folder (folder bisa pakai opsi -r). 

Windows (Command Prompt)-- perintah akses/manipulasi file/folder:
dir - lihat isi folder/direktori. 
cd - berubah direktori (atau folder) di drive. 
mkdir atau md - buat folder baru. 
del - hapus file. 
copy atau xcopy - salin file / folder. (Contoh xcopy spesial)

6. “Terminal” biasanya istilah yang dipakai di sistem Unix/Linux (atau Mac) — aplikasi untuk memasukkan perintah teks ke sistem operasi, sedangkan “Command Prompt” (atau CMD) khusus di Windows adalah jenis terminal command-line khusus Windows.
